JXZ

Salta se il registro rX è uguale a zero.

Codice operativo: 47

N.B.: Il codice operativo 47 è condiviso con le altre istruzioni di salto con condizioni basate sul contenuto del registro rX: JXE, JXN, JXNN, JXNZ, JXNP, JXO e JXP.

Modificatore: 01

Il modificatore, con valore fisso pari a 01, fa parte integrante della specifica di questa operazione.

Descrizione

La condizione di questa istruzione di salto è che il contenuto del registro rX sia uguale a zero. Se tale condizione è vera, l'indirizzo dell'istruzione successiva viene caricato nel registro di salto rJ, e l'esecuzione salta all'indirizzo dell'operando M.

In simboli: se rX = 0,

rJ ← P,
P ← M,
dove P rappresenta l'indirizzo della prossima istruzione che deve essere eseguita.

Esempio

Il programma

INIZIO ENT1 0
CERCA LDX TABELLA,1
JXZ TROVATO
INC1 1
JMP CERCA
TROVATO HLT
TABELLA CON 10
CON 0
CON -8
END INIZIO

termina con rI1 = 1.
