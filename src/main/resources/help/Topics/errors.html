Messaggi d'errore

Di seguito sono riportati i messaggi d'errore che possono essere riportati dall'ambiente MixIDE, divisi per categorie.

Messaggi d'errore generati dall'assemblatore

Unexpected end of file.

Il file sorgente è terminato mentre l'assemblatore si aspettava di trovare qualcosa oppure il programma non è stato terminato, come si deve sempre fare, dalla direttiva END.
