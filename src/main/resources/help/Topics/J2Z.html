J2Z

Salta se il registro rI2 è uguale a zero.

Codice operativo: 42

N.B.: Il codice operativo 42 è condiviso con le altre istruzioni di salto con condizioni basate sul contenuto del registro rI2: J2N, J2NN, J2NZ, J2NP e J2P.

Modificatore: 01

Il modificatore, con valore fisso pari a 01, fa parte integrante della specifica di questa operazione.

Descrizione

La condizione di questa istruzione di salto è che il contenuto del registro rI2 sia uguale a zero. Se tale condizione è vera, l'indirizzo dell'istruzione successiva viene caricato nel registro di salto rJ, e l'esecuzione salta all'indirizzo dell'operando M.

In simboli: se rI2 = 0,

rJ ← P,
P ← M,
dove P rappresenta l'indirizzo della prossima istruzione che deve essere eseguita.

Esempio

Il programma

EL2 CON EL3(0:2)
CON 5
EL3 CON 0
CON 3
EL1 CON EL2(0:2)
CON 10
INIZIO ENT2 EL1
ENTA 0
CICLO J2Z FINE
SUM 1,2
LD2 0,2(0:2)
JMP CICLO
FINE HLT
END INIZIO

segue la lista collegata di tre elementi (EL1, EL2, EL3) e ne somma i dati (10, 5, 3), terminando con rA = 18.
