JAN

Salta se il registro rA è negativo.

Codice operativo: 40

N.B.: Il codice operativo 40 è condiviso con le altre istruzioni di salto con condizioni basate sul contenuto del registro rA: JAE, JANN, JANZ, JANP, JAO, JAP e JAZ.

Modificatore: 00

Il modificatore, con valore fisso pari a 00, fa parte integrante della specifica di questa operazione.

Descrizione

La condizione di questa istruzione di salto è che il contenuto del registro rA sia strettamente minore di zero. Se tale condizione è vera, l'indirizzo dell'istruzione successiva viene caricato nel registro di salto rJ, e l'esecuzione salta all'indirizzo dell'operando M.

In simboli: se rA < 0,

rJ ← P,
P ← M,
dove P rappresenta l'indirizzo della prossima istruzione che deve essere eseguita.

Esempio

Il programma

INIZIO ENT1 0
CERCA LDA TABELLA,1
JAN TROVATO
INC1 1
JMP CERCA
TROVATO HLT
TABELLA CON 10
CON 0
CON -8
END INIZIO

termina con rI1 = 2.
